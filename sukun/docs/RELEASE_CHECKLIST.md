# قائمة الإصدار

## حواجز إلزامية

- [ ] Family Controls distribution entitlement موافق عليه للأربعة App IDs.
- [ ] App Group متطابق في الملفات والبوابة وملفات provisioning.
- [ ] Archive Release ينجح بلا تحذيرات entitlement أو extension point.
- [ ] اختبار monitor start/end على جهازي iPhone وإصدارَي iOS مدعومين.
- [ ] اختبار رفض الصلاحية، إلغائها من Settings، واختيار فارغ.
- [ ] مراجعة مواقيت 30 يومًا مقابل مصدر محلي موثوق في مدن الإطلاق.
- [ ] إضافة offsets لكل مدينة إذا تجاوز الفرق دقيقتين.
- [ ] VoiceOver، Dynamic Type، contrast، وRTL مكتملة.
- [ ] App Privacy في App Store Connect يطابق Privacy Manifest: لا تتبع ولا جمع بيانات.
- [ ] لقطات App Store عربية وإنجليزية، وصف واضح لحدود Screen Time.
- [ ] سياسة خصوصية ودعم منشوران على URLs عامة.

## TestFlight

- [ ] Internal test: 7 أيام، لا missed callbacks غير مفسّرة.
- [ ] External test: 25 مستخدمًا، ≥95% من اختبارات الصحة تصبح Ready.
- [ ] جمع البلاغات يدويًا فقط، من دون SDK تحليلات في الإصدار الأول.
